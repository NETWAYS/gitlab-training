# Git Server Protocol

* Read/write access via SSH
  * `git@github.com:username/repo.git`
* HTTPS protocol (write access via oauth tokens)
  * `https://my-gitlab.nws.netways.de/username/repo.git`
* Git protocol
  * `git://domain.com/repo.git`
* Local protocol
  * `file:///opt/git/repo.git`
